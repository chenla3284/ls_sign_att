# ls_sign_att
自動簽到吧

1. 須安裝 google-chrome、chromedriver
  ```
  [EXAMPLE for MAC]
  brew install --cask google-chrome
  brew install --cask chromedriver
  ```

2. 安裝需求套件
  ```
  pip3.9 install -r requirements.txt
  ```
  ```
  (pip install -r requirements.txt)(optional)
  ```
 
3. 設定 .env
  ```
  參考 .env.example
  ```

4. 執行
  ```
  python3.9 Main.py
  ```
